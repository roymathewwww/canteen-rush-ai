# Canteen Rush AI

A high-fidelity, autonomous pre-order and queue management system for university campuses.

## Project Structure

- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4 (Zinc/Neutral palette)
- **Fonts**: Manrope (UI), JetBrains Mono (Data)
- **Icons**: Lucide React

## Getting Started

1.  Navigate to the project directory:
    ```bash
    cd canteen-rush-ai
    ```

2.  Install dependencies (if not already done):
    ```bash
    npm install
    ```

3.  Run the development server:
    ```bash
    npm run dev
    ```

4.  Open [http://localhost:3000](http://localhost:3000) with your browser to see the Student View.
5.  Open [http://localhost:3000/vendor/dashboard](http://localhost:3000/vendor/dashboard) to see the Vendor Dashboard.

## Features

-   **Student Context Awareness**: Auto-predicts "Can I eat?" based on time.
-   **Live Order Prediction**: Calculates pickup time based on cart complexity.
-   **Tokens**: Anxiety-reducing status screens.
-   **Vendor Dashboard**: Peripheral-vision optimized queue management.
